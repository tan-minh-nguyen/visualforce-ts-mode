* visualforce-ts-mode

Tree-sitter based major mode for Salesforce Visualforce pages in Emacs.

** Features

- Syntax highlighting via tree-sitter
- Indentation support
- Imenu navigation for Visualforce components
- LSP integration (eglot, lsp-bridge)

** Requirements

- Emacs 29.1+ (with tree-sitter support)
- tree-sitter-html grammar (Visualforce is XML-like)

** Installation

*** Grammar Installation

#+begin_src emacs-lisp
(add-to-list 'treesit-language-source-alist
             '(html "https://github.com/tree-sitter/tree-sitter-html"))
(treesit-install-language-grammar 'html)
#+end_src

*** Package Installation

Add to your load-path:

#+begin_src emacs-lisp
(add-to-list 'load-path "/path/to/visualforce-ts-mode")
(require 'visualforce-ts-mode)
#+end_src

** Usage

#+begin_src emacs-lisp
(require 'visualforce-ts-mode)
;; .page and .component files automatically use visualforce-ts-mode
#+end_src

** Extensions

| Extension | Purpose | Requires |
|-----------+---------+----------|
| visualforce-eglot | Eglot LSP integration | eglot |
| visualforce-lsp-bridge | LSP-bridge integration | lsp-bridge |

** File Associations

#+begin_src emacs-lisp
(add-to-list 'auto-mode-alist '("\\.page\\'" . visualforce-ts-mode))
(add-to-list 'auto-mode-alist '("\\.component\\'" . visualforce-ts-mode))
#+end_src
